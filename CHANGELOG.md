# Changelog

## v1.0.0 — Initial Release

### Features
- Full Nextflow DSL2 pipeline with 12 modular steps
- Conda, Docker, Singularity, SLURM, PBS, AWS, GCP support
- Splice-aware HISAT2 alignment for eukaryotic transcriptomes
- Per-nucleotide coverage assessment and targeted masking
- BCFtools SNP calling with quality filtering
- Collapse and Retain heterozygosity strategies
- MAFFT + IQ-TREE2 phylogenetic inference
- Robinson-Foulds distance + Kolmogorov-Smirnov bootstrap comparison
- Interactive self-contained HTML report
- Publication-ready PDF figures (topology + bootstrap)
